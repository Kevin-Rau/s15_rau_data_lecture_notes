Web Servers

  - GET/POST/DELETE/PUT

  - Uploads of prodcuts or likes are calls to Web Servies handled in JSON or XML,

  - /users |GET -> READ | POST -> CREATE | PUT -> UPDATE | DELETE -> DESTROY |
